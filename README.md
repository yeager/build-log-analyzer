# Build Log Analyzer

Analyze sbuild and pbuilder build logs with error highlighting.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install build-log-analyzer
```
